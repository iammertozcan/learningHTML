HTML Nedir?

HTML, High Text Markup Language kelimelerinin kısaltmasıdır. HTML komutları, metin, resim, ses, video gibi içeriklerin sayfadaki yerleşimini ve ziyaretçiye gösterilmesini sağlar. HTML’in en önemli özelliklerinden biri sayfanın bir yerinden başka bir yerine veya başka bir sayfaya gitmemizi sağlayan bağlantılarıdır.

HTML Sayfası Nedir?

HTML dili, adına tag ya da etiket denen komutlardan oluşur. Bu komutlar bir araya gelerek web/html sayfasını oluşturur. HTML sayfalarının dosya uzantısı .htm veya .html’dir.

HTML Sürümleri

HTML dili 1980 yılında CERN laboratuvarlarında görevli olan Tim Berners-Lee tarafından geliştirilmiştir. Lee’nin amacı CERN’de çalışan araştırmacıların bilgi ve dökümanlarını kolayca paylaşılabilmesi için bir sistem oluşturmaktı. 1980 yılında CERN laboratuvarında kıllanılmaya başlanan HTML, yeni sürümleriyle güncel teknolojilere uyum sağlayarak günümüze kadar gelmiştir.

HTML Editörleri

Kod yazmaya yarayan yazılımlara editör denir. Piyasada yaygın olarak aşağıdaki HTML editörleridir:

  1. Visual Studio
  2. Visual Studio Code
  3. Dreamweaver
  4. Notepad++
  5. Atom
  6. Sublime
  7. Netbeans
  8. Brackets

Tarayıcı Nedir?

Tarayıcılara (browser) sayfa içeriğini HTML kodlarına göre işleyip düzenleyerek (render) kullanıcıya gösteriyor.

Ağ Nedir?

Herhangi bir evde veya işyerinde birbirie bağlanan bilgisayarlar bir ağ oluştururlar. Bu tür ağlara Yerel Ağ (LAN) denir. Yerel ağlar birbirine bağlanarak Geniş Ağ (WAN), geniş ağlar da birbirine bağlanarak dünya çağında bir ağ olan World Wide Web’i (WWW) oluştururlar.

Web Sunucu Nedir?

HTML sayfalarını başka kişilerin de ziyaret etmesini istiyorsanız bu sayfaları Web Sunucu denilen bilgisayarlara koymalısınız.
